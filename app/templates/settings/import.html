Инструкция
Порядок импорта:
  1. Скачайте шаблон — нажмите "Скачать шаблон" для нужного типа данных
  2. Заполните данные — откройте JSON в текстовом редакторе и добавьте свои данные по образцу
  3. Сначала справочники — импортируйте справочники (создаёт группы, преподавателей и т.д.)
  4. Затем ГУП — импортируйте ГУП (привязывает недели к группам)
Важно!
  • Файлы должны быть в формате JSON (кодировка UTF-8)
  • Названия групп в ГУП должны точно совпадать с названиями в справочниках
  • Существующие записи не дублируются (проверка по имени/коду)
