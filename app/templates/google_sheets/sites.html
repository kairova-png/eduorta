Информация

GitHub Pages — это сервис для публикации веб-сайтов прямо из GitHub репозитория.

Что происходит при синхронизации:

  • Генерируются HTML файлы с расписанием
  • Создается JSON API для расписания
  • Файлы отправляются в GitHub репозиторий
  • Сайт автоматически обновляется
Совет: Используйте автосинхронизацию для автоматического обновления сайта после генерации.
