## 🎯 КАК ЗАПУСТИТЬ В ПРОДАКШНЕ:

### Способ 1: Простой запуск
```bash
set FLASK_ENV=production
python run.py
```

### Способ 2: Windows батник (рекомендуется)
```bash
start_production.bat
```

### Способ 3: С Gunicorn (лучшая производительность)
```bash
pip install gunicorn
gunicorn --bind 127.0.0.1:5000 --workers 2 wsgi:app
```

## 📁 Структура файлов:

```
college_schedule/
├── run.py                  # Главный файл запуска  
├── wsgi.py                # WSGI entry point
├── config.py              # Продакшн конфигурация
├── requirements.txt       # Зависимости
├── .env.example           # Пример переменных окружения
├── start_production.bat   # Скрипт запуска для Windows
├── app/                   # Основное приложение
├── instance/              # База данных
├── logs/                  # Логи (создается автоматически)
└── exports/               # Экспорты
```

## 🔒 Переменные окружения:

Скопируйте `.env.example` в `.env` и настройте:

```
FLASK_ENV=production
SECRET_KEY=your-secure-key-here
DATABASE_URL=sqlite:///instance/college.db
LOG_LEVEL=INFO
```

## 📊 Логирование:

- Логи сохраняются в папку `logs/`
- Ротация файлов: 10MB, 5 бэкапов
- Уровень логирования настраивается через LOG_LEVEL
