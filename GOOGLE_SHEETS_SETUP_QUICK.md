# Быстрая настройка Google Sheets интеграции

## 🚀 Пошаговая инструкция

### Шаг 1: Создание Google Apps Script (2 минуты)

1. Перейдите на https://script.google.com
2. Нажмите "Новый проект"
3. Скопируйте весь код из файла `google_apps_script/webhook_receiver.gs` 
4. Вставьте код в редактор, заменив весь существующий код
5. Нажмите **Ctrl+S** для сохранения

### Шаг 2: Создание Google Таблицы (1 минута)

1. Перейдите на https://sheets.google.com
2. Нажмите "Создать" → "Пустая таблица"
3. Переименуйте таблицу в "Расписание APEC"
4. Скопируйте ID таблицы из URL (длинная строка между `/d/` и `/edit`)

**Пример URL:** https://docs.google.com/spreadsheets/d/**1A2B3C4D5E6F7G8H9I0J**_/edit  
**ID таблицы:** 1A2B3C4D5E6F7G8H9I0J

### Шаг 3: Настройка Script (1 минута)

1. В Google Apps Script найдите строку:
   ```javascript
   SPREADSHEET_ID: 'YOUR_SPREADSHEET_ID_HERE',
   ```
2. Замените `YOUR_SPREADSHEET_ID_HERE` на скопированный ID таблицы
3. Сохраните (**Ctrl+S**)

### Шаг 4: Развертывание Script (2 минуты)

1. Нажмите **"Deploy"** (Развернуть) → **"New deployment"**
2. Выберите тип: **"Web app"**
3. Настройки:
   - Execute as: **"Me"** 
   - Who has access: **"Anyone"**
4. Нажмите **"Deploy"**
5. **Скопируйте URL** веб-приложения (начинается с https://script.google.com/macros/s/...)

### Шаг 5: Настройка Flask приложения (30 секунд)

**Вариант A: Переменная окружения (рекомендуется)**
```bash
export GOOGLE_SHEETS_WEBHOOK_URL="ВСТАВЬТЕ_СЮДА_URL_ИЗ_ШАГА_4"
```

**Вариант B: Файл config.py**
Добавьте в config.py:
```python
GOOGLE_SHEETS_WEBHOOK_URL = 'ВСТАВЬТЕ_СЮДА_URL_ИЗ_ШАГА_4'
```

### Шаг 6: Настройка доступа к таблице (30 секунд)

1. В Google Таблице нажмите **"Share"** (Настроить доступ)
2. Выберите **"Anyone with the link can view"**
3. Нажмите **"Copy link"** для получения публичной ссылки

## ✅ Готово! 

Теперь при генерации расписания данные автоматически отправляются в Google Таблицу.

### Проверка работы:

1. Перейдите в приложении в раздел **"Google Sheets"** 
2. Нажмите **"Тест подключения"**
3. Если подключение успешно - нажмите **"Синхронизировать все"**

### Публичная ссылка на расписание:

Отправьте студентам ссылку на Google Таблицу для просмотра актуального расписания.

## 🆘 Решение проблем

**Ошибка при тесте подключения:**
- Проверьте правильность URL webhook'а
- Убедитесь что Script развернут с доступом "Anyone"

**Данные не появляются в таблице:**
- Проверьте ID таблицы в Google Apps Script
- Убедитесь что у Script есть права на редактирование таблицы
